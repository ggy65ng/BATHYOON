# Bathyoon Store landing page

Static bilingual landing page for Bathyoon Store (digital game services). Includes language toggle (AR/EN), theme switcher (dark/light), and currency preview (SAR/USD/EUR) with RTL/LTR support. Ready to deploy on Netlify.

## Getting started

Open `index.html` in a browser or serve locally (e.g. `npx serve .`). Netlify can deploy directly from this repository.

### Controls
- Language: top-right AR/EN button, updates all text and direction.
- Theme: moon/sun button toggles dark and light modes (remembered in local storage).
- Currency: SAR/USD/EUR selector updates the informational prices on service cards.
